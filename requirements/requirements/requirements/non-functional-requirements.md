# Requerimientos No Funcionales - RecetaHub

1. **Accesibilidad (Estándar WCAG 2.1 nivel AA)**
   - La plataforma debe garantizar una relación de contraste mínima de 4.5:1 en todos los textos para asegurar la legibilidad por personas con baja visión.
   - Todos los elementos de la interfaz y menús de navegación deben ser operables mediante el uso exclusivo del teclado.

2. **Rendimiento y Carga**
   - El tiempo de respuesta del sistema para cargar el catálogo de recetas y el planificador semanal no debe superar los 4 segundos bajo condiciones normales de velocidad de internet.

3. **Compatibilidad de Navegadores**
   - El sistema debe funcionar de forma optima en las versiones más recientes de los navegadores Google Chrome y Microsoft Edge.

4. **Seguridad y Sesiones**
   - El sistema debe realizar el cierre automático de sesiones tras 30 minutos de inactividad para proteger la integridad de las cuentas de usuario.
