# Requerimientos Funcionales - RecetaHub

* **RF-01: Registro de Nueva Receta (Chef)**
  El sistema permite al Chef crear recetas completas ingresando: nombre, tipo de plato, dificultad, tiempo de preparación y porciones. Incluye una sección para subir imágenes, una tabla dinámica para ingredientes con su costo individual y una lista de pasos numerados.

* **RF-02: Catálogo de Recetas con Filtros (Usuario)**
  Interfaz de galería para explorar todas las recetas disponibles. Permite filtrar la búsqueda mediante una barra superior y filtros laterales por categoría (Desayuno, Almuerzo, Cena, Postres) y nivel de dificultad (Fácil, Intermedio, Avanzado).

* **RF-03: Detalle de Receta y Feedback (Usuario)**
  Pantalla que muestra la información completa de una receta, incluyendo ingredientes y procedimiento. Permite al usuario calificar la preparación mediante estrellas, dejar comentarios de texto y guardar la receta en colecciones personalizadas.

* **RF-04: Planificador Semanal (Usuario)**
  Herramienta de calendario (Lunes a Domingo) donde el usuario asigna recetas a los diferentes tiempos de comida. El sistema calcula automáticamente el presupuesto total estimado basado en las recetas seleccionadas para la semana.

* **RF-05: Lista de Compras Inteligente (Usuario)**
  Módulo que genera automáticamente un listado de ingredientes agrupados por categorías (Proteínas, Verduras, Lácteos, etc.) a partir de lo planificado. Incluye casillas de verificación (check-box) para marcar los productos ya obtenidos.

* **RF-06: Panel de Gestión del Chef (Chef)**
  Perfil privado donde el profesional puede administrar sus publicaciones, ver el estado de sus recetas (Publicadas o Borradores), monitorear su número de seguidores y las valoraciones promedio recibidas.
