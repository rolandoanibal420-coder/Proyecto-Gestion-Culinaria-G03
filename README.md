# Proyecto: RecetaHub - Grupo 03

## Descripción y Alcance
RecetaHub es una plataforma diseñada para la gestión integral de recetas culinarias. Permite a Chefs administrar contenido profesional y a usuarios regulares buscar recetas, organizar menús semanales y generar listas de compras.

## Estrategia de Commits
Para mantener la organización del repositorio, el equipo utilizará los siguientes prefijos en los mensajes de commit:
* **docs:** Actualizaciones en la documentación y archivos Markdown.
* **feat:** Registro de nuevas funcionalidades del sistema.
* **fix:** Corrección de errores detectados.
* **style:** Cambios en el diseño, interfaces o formato visual.
* **new:** Cuando ingresamos nueva informacion
